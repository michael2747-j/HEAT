🔥 H.E.A.T. - Header-based Ether Adapter Tracking

Automated VyOS configuration generation from live network traffic.

🌐 What Is H.E.A.T.?

H.E.A.T. is a cross-platform system designed to transform any device — desktop or mobile — into a network configuration wizard.

  • Scan real network traffic
  • Parse DNS, VLAN, BGP, DHCP, VPN, and more
  • Generate secure, deployable VyOS configurations
  • Push configs to routers in one click

🎯 Project Goal

“Turn smartphones and PCs into plug-and-play network config wizards.”

With real-time packet capture, HEAT builds tailored VyOS configs in under 2 minutes — no CLI needed.

🧱 Core Architecture

  • 🖥 Windows App – built with WinUI 3 and .NET
  • 📱 Android App – Kotlin/NDK-based mobile interface
  • ⚙️ C++ Modules – fast native packet parsers
  • 🛠 VyOS Config Generator – config output layer
  • 📤 SSH Export – deploy config using libssh

💡 Key Requirements (from David)

  • 🕐 Fast: generate basic configs in under 2 mins
  • 🔍 Capture: DNS, DHCP, VLAN, NAT, AD controllers
  • ✅ Export: ready-to-use VyOS config file
  • 🧪 Test: virtualized networks with VyOS in Proxmox
  • 🛡️ Security: detect anomalies, alert users
  • 📱 Extensible: Windows + Android now, iOS later

🔀 Branch Overview

  • main – 📘 Docs, roadmap, coordination
  • windows-app – 🖥 Desktop implementation
  • android-app – 📱 Mobile interface
  • cpp-modules – ⚙️ Shared packet parsers

📦 Features

  • Scan ➝ Parse ➝ Generate Config
  • Detect services: DNS, VPN, VLAN, BGP, NAT
  • Preview configs in the UI before exporting
  • Security alerting & logging
  • Push config to router via SSH
